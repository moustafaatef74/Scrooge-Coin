1)open eclipse 
2)click on file
3)click on import
4)click on General
5)click on Archive File
6)Choose this zip File

To start the simulation:

run Scrooge class: the users will be created as every user get assigned 10 coins and then the user's publicKey and number of coin will be printed

to Simulate 5 valid transactions press A

to Simulate a double spending attack press S

to Simulate an invalid signature transaction press D

to Simulate an invalid owner transaction  press F

to quit press SPACE

the output will be in Project\ScroogeCoin\src\Output.txt containing the users with their initial amounts of coins then the simulation output
